Linux学习之bash shell
2020-03-21

Linux支持的shell

  • Linux支持的shell在/etc/shells文件中
  • Linux使用的版本称为Bourne Again SHell(bash)
  • BASH主要优点
    • 命令记忆能力
    • 档案比对补全功能
    • 命令别名设定功能（alias）
    • 作业控制、前台后台控制
    • shell script的强大功能
    • 通配符
  • type [-ta] name
    • 不加任何参数时，type显示出name是外部命令还是bash内置的命令
    • -t:通过文字显示：file、alias、builtin
    • -a:在PATH变量定义的路径中，列车所有含有name的命令，包含alias

Shell的变量功能

  • echo $变量名
  • 在Linux预设的情况中，使用大写的字母来设定的变量一般为系统预定义的变量
  • env显示目前系统中主要的预定义环境变量内容
  • set可以将环境变量和用户自定义变量显示出来

PS1提示符的设置

  • \d:代表日期，格式为星期 月 日
  • \t:24小时格式显示时间，HH:MM:SS
  • \A:24小时格式显示时间，HH:MM
  • \T:12小时格式显示时间，HH:MM:SS
  • \H:完整的主机名称
  • \h:只取主机名称的第一个名字
  • \u:当前账户名
  • \v:bsah版本信息
  • \W:只列出最后一个目录
  • \w:完整工作目录
  • #:执行的第几个命令
  • $:root用户提示符是#,其他用户提示符是$

export

  • export:将用户自定义变量变为环境变量
  • 在启动shell的时候，系统给shell分配一块内存，用来存储系统环境变量
  • export可以将用户自定义变量移动到该内存中
  • 当进入子shell时，子shell将父shell的系统环境变量读入到子shell的内存中

bash下变量设定的规则

  • 变量与变量内容以等号=来连结，等号两边不能直接接空格符
  • 变量名称只能是英文字母与数字，但是数字不能是开头字符
  • 若有空格符可以使用双引号或单引号来将变量内容结合起来，但要注意，双引号内的特殊字符可以保持原有变量特性，但是单引号内的特殊字符只是一般的字符
  • 转义字符\
  • 在一串指令时，如果还需要由其他指令提供的信息，可以使用quote [‘command’]
  • 若该变量为扩增变量内容时，则需要以双引号和$变量名称,比如：[PATH=”$PATH”:/home] 继续累加变量内容
  • 取消变量：[unset 变量名称]

一些定义

  • 语系文件的变量locate
  • locate -a显示系统支持的所有语系
  • 改变系统支持的语系，可修改/etc/sysconfig/i18n

变量键盘读取和声明变量

  • read [-pt] 变量
    • -p:后面可以接提示符
    • -t:后面可以接秒数，限制等待的时间
  • declare/typeset [-airx] 变量
    • -a:声明的变量为数组
    • -i:整数数字
    • -r:变量为只读
    • -x:变量变成系统环境变量
    • 数组：a[i]

命令别名与历史命令

  • alias
  • history (history n) [-crw]
    • !的用法，快捷使用过往命令

Bash shell的使用环境

  • /etc/issue：系统信息
  • /etc/motd：登陆时系统读给用户的信息
  • /etc/profile,etc/bashrc:系统设定值
    …

个人设定值

  • ~/.bash_profile
  • ~/.bashrc
  • ~/.bash_logout
  • source

Bash shell使用环境的终端的环境设置

  • stty -a
  • 通配符*与?的区别：*可表示多个字符
  • 引用：使用双括号或单引号，引用内容先计算

数据流重定向

  • 就是将目前的数据输出到其他地方去
    • 标准输出：1> >>
    • 标准输入: < <<
    • 标准错误输出: 2> >>
  • 写入同一个档案推荐使用：find / -name file 1> list 2>&1
  • 1>> 2>> ：累加的方式输出
  • 双向重定向：tee
    • 让standard output转存一份到文件内并将同样的数据继续送到屏幕去处理

管道命令

  • 只能处理前面一个命令传来的正确信息
  • 选取命令：cut,grep
    • 将一段数据经过分析后，取出想要的部分,以行为单位
    • cut -d ‘分割字符’ -f fields
    • grep [-acinv] ‘搜索字符串’ filename
    • grep取得关键字，将整行列出来
  • 排序命令：sort,wc,uni
    • sort [-fbMnrtuk] [file or stdin]
    • uniq [-ic]
    • wc [-lwm]
  • 字符转换命令：tr,col,join,paste,expand
    • tr [-ds] str… ：删除一段信息当中的文字，或进行替换
    • col [-x]
    • expand [-t] file
    • join [-ti12] file1 file2
    • paste [-d] file1 file2
  • 切割命令：split
    • split [-bl] file PREFIX
  • 参数代换：xargs
    • xargs [-epn] command
    • …
