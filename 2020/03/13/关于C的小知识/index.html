对于C的一次重新学习的知识点归纳

  • 1.const:定义常量

  • 2.数组函数

    • 不能在[]中给出数组大小
    • 不能用sizeof计算数组元素个数
    • 如：int search(int key, int a[], int length)

  • 3.指向结构的指针

    • 用->表示指针所指的结构变量的成员
    • 用.表示非指针引用

  • 4.自定义数据类型

    • 如：typedef int Length

  • 5.联合union

    • sizeof(union …) = sizeof(每个成员)的最大值

  • 6.静态本地变量

    • 在本地变量定义时加上static修饰符
    • 初始化只会在第一次进入函数时
    • 当函数离开时，继续存在并保持其值

  • 7.sizeof:静态的，sizeof(a++);a值不会计算。

  • 8.编译预处理指令

    • 以#开头
    • 不是C语言成分
    • #define用来定义一个宏

  • 9.#define

    • #define <名字> <值>
    • 无分号，不是C语句
    • 名字必须是一个单词，值可以是各种数据
    • 编译预处理程序把程序中的名字替换成值

  • 10.带参数的宏的原则

    • 整个值要括号
    • 参数出翔的地方要括号
    • 如：#define MIN(a,b) ((a)>(b)?(b):(a))

  • 11.头文件

    • 把函数原型放在一个文件(.h),在需要调用这个函数的源代码文件(.c)中#include这个头文件
    • #include不是引入库的
    • stdio.h里只有printf的原型
    • #include<stdio.h>只是为了让编译器知道printf函数的原型，保证调用时给出的参数值是正确的类型。
    • 在使用和定义这个函数的地方都应该#include这个头文件
    • 一般的做法是任何.c都有对应的同名的.h,把所有对外公开的函数的原型和全局变量的声明放进去
    • static可以使函数和全局变量只在所在编译单元使用
    • extern int i;变量的声明 ， int i;变量的定义
    • 同一个编译单元里，同名的结构不能被重复声明
    • 标准头文件结构
      #ifndef _HEAD_
      #define _HEAD_
      ...
      #endif

  • 12.格式化输入输出
    %[flags][width][.prec][hlL]type

    • Flag:-:左对齐，+:在前面加+或-，(space):正数留空,00填充。
    • width或prec：number:最小字符数，*:下一个参数是字符数，.number:小数点后位数，.*:下一个参数是小数点的位数

    scanf:%[flag]type
    *:跳过，数字:最大字符数，hh:char,h:short,l:long,double,ll:long long,L:long double。
