Linux学习之shell script
2020-04-05

1.基础概念

  • 针对shell所写的脚本
  • 程序编写第一行：#!/bin/bash,声明这个script使用的shell名称
  • 主要环境变量的声明：
    PATH=/bin:/sbin:/usr/bin:/usr/sbin:/usr/local/bin:/usr/local/sbin:~/bin
    export PATH
  • 告知执行结果：exit 0
  • 数值运算：$((计算式)),运算处理有：+、-、*、/、%

2.shell script

  • script的执行方式的区别
    • 直接执行(sh、bash)：实际执行的脚本在子进程中，变量执行完后即删除
    • source执行：脚本在父进程中执行
  • test命令的测试功能
    • 关于某个文件名的“文件类型”判断，如：test -e filename表示存在否
    • 关于文件的权限检测
    • 两个文件的比较,如test file1 -nt file2
    • 关于两个整数之间的判定，如：test n1 -eq n2
    • 判定字符串的数据
    • 多重条件判定，如：test -r filename -a -x filename
  • 判断符号[]
    • 在[]内的每个组件都要空格键来分隔
    • 在[]内的变量，最好都要双引号括起来
    • 在[]内的常量，最好都要单引号或双引号括起来
  • shell script的默认变量
    • /path/to/scriptname opt1 opt2 opt3 opt4
             $0                      $1    $2    $3     $4
    • $#:后接的参数的个数
    • $@:代表”$1”、”$2”、”$3”、”$4”，每个变量是独立的
    • $*:代表“ “$1c$2c$3c$4” ”,其中c为分割字符，默认为空格键
    • shift:造成参数变量的偏移，后接数字，代表拿掉几个最前面的参数
